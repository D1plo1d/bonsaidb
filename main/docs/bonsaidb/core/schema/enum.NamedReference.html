pub enum NamedReference<'a, Id> {
    Name(Cow<'a, str>),
    Id(DocumentId),
    Key(Id),
}
Expand description

A reference to a collection that has a unique name view.

Variants

Name(Cow<'a, str>)

An entity’s name.

Id(DocumentId)

A document id.
